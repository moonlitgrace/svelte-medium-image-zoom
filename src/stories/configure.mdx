# Introduction

This site showcases different usages of
[`svelte-medium-image-zoom`](https://github.com/moonlitgrace/svelte-medium-image-zoom),
the original medium.com-inspired image zooming library for Svelte.\
Use the links in the sidebar to try out different features that the library
supports.

For more info, please consult the [`svelte-medium-image-zoom`
README](https://github.com/moonlitgrace/svelte-medium-image-zoom#readme).
